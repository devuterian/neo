# Research records

`records/research/` stores curated, reusable technical research for repository development. It is not a transcript archive, execution log, message-log mirror, or personal record store.

Research notes should:

- state the repository question being answered;
- cite public or repository-local sources used;
- separate confirmed facts from recommendations;
- omit credentials, token values, private records, and personal operational data;
- link to related ADRs, plans, or PRs when a conclusion becomes accepted work.
