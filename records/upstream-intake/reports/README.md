# Upstream intake reports

Store concrete `LPFchan/repo-template` review reports here when an upstream change is evaluated. Reports should be concise and should record:

- upstream reference reviewed;
- local files or contracts affected;
- decision: adopt, adapt, defer, or reject;
- rationale;
- follow-up PR or issue if needed.

Do not paste secrets, personal operational records, raw message logs, or full upstream scaffold copies into reports.
