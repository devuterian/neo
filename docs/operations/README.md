# Operations documentation

This directory is the safe repository entry point for operating `neo` as software and as an OCI-backed Hermes workspace. It summarizes confirmed repository facts and links to existing detailed documentation instead of duplicating long procedures.

- [`AGENT_MODES.md`](AGENT_MODES.md): Hermes runtime and repository-development read boundaries.
- [`DEVELOPMENT.md`](DEVELOPMENT.md): branch, PR, validation, and documentation-change workflow.
- [`OCI.md`](OCI.md): OCI live runtime clone boundaries.
- [`DEPLOYMENT.md`](DEPLOYMENT.md): merge-to-live deployment expectations and limits.
- [`integrations/`](integrations/): external integration notes confirmed by repository documentation.

Do not copy `.env` contents, tokens, credentials, private records, message-log contents, life-day notes, or exported personal records into these pages.
