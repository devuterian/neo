# BeeperBox

BeeperBox is the documented path for reading and sending messages through external messengers such as Discord or Instagram when that capability is available through the local MCP server.

## Confirmed repository facts

- The root repository instructions refer to BeeperBox MCP at `localhost:23375` for external messenger messages.
- Messenger contents are personal operational data. Do not copy message contents, credentials, tokens, or session data into repository documentation.

## Use boundary

Use BeeperBox only for the user-requested messenger action and follow any more specific skill or tool instructions available in the active environment. Keep message content out of changelog and repository records unless the user explicitly asks for a safe, non-sensitive repository-development note.
