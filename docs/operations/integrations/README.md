# Integrations

This directory summarizes external integration operations confirmed by repository documentation. Prefer links to the detailed source sections in the root README rather than duplicating long procedures that can drift.

- [`kakao-map.md`](kakao-map.md): Kakao place search MCP wrapper.
- [`daiso-mcp.md`](daiso-mcp.md): local retail, stock, place, fuel, and cinema lookup via remote MCP / CLI.
- [`beeperbox.md`](beeperbox.md): external messenger access boundary.

Integration outputs can be time-sensitive. When reporting place, transport, stock, or schedule results, state that they are based on the lookup time.
